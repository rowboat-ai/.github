# Rowboat AI

AI-powered group outing coordination.

Rowboat helps groups plan outings end-to-end by combining preferences, calendars, venue discovery, and recommendation ranking into one workflow.

## What We’re Building

- AI-assisted group planning
- venue and event discovery
- shared preference collection
- calendar-aware coordination
- explainable recommendations
